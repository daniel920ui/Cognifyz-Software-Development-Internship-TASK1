# Task 1 - Number Guessing Game

## Objective
Develop a simple text-based game using Java.

## Tools Used
- Java
- Scanner
- Random

## Description
The program generates a random number between 1 and 100. The user keeps guessing until the correct number is found.

## Features
- Random number generation
- User input handling
- Too High / Too Low hints
- Attempt counter

## Concepts Used
- Loops
- Conditional Statements
- Java Classes

## Output
The game displays whether the guess is too high, too low, or correct.
