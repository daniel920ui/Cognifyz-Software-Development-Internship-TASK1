# Task 4 - Temperature Converter

## Objective
Develop a temperature converter that converts temperatures between Celsius and Fahrenheit.

## Tools Used
- Java
- Scanner Class

## Features
- Celsius to Fahrenheit Conversion
- Fahrenheit to Celsius Conversion
- User-Friendly Menu

## Formula Used

Celsius to Fahrenheit:
F = (C × 9/5) + 32

Fahrenheit to Celsius:
C = (F - 32) × 5/9

## Concepts Used
- Conditional Statements
- User Input Handling
- Mathematical Calculations

## Description
The program allows users to choose a conversion type and enter a temperature value. The converted result is then displayed.

## Outcome
Successfully implemented a temperature conversion application using Java.
